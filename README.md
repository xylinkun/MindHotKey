# MindHotKey - 中文文本整理与思维导图辅助工具

一款基于 AutoHotkey 的高效中文文本整理与思维导图辅助工具，通过快捷键快速规范剪贴板中的字符串并规范标点，生成适合构建思维导图的结构化内容。

---

## 📖 项目简介

MindHotKey 是一款专为中文文本处理设计的辅助工具，特别适合需要频繁整理文本、制作思维导图的用户。它能够：

- ✅ 自动修正 OCR 识别文本中的常见错误
- ✅ 统一标点符号格式（全角/半角自动转换）
- ✅ 智能删除多余的空格和换行
- ✅ 繁体字转简体字
- ✅ 中文数字转阿拉伯数字
- ✅ 按标点符号自动分段
- ✅ 添加序号、整理列表
- ✅ 快速输入常用符号和表情

---

## 🚀 快速开始

### 1️⃣ 安装 AutoHotkey

1. 访问 [AutoHotkey 官网](https://www.autohotkey.com/)
2. 下载并安装 **AutoHotkey v2**（必须是 v2 版本）

### 2️⃣ 运行脚本

1. 双击 `MindMap.ahk` 文件运行
2. 脚本会自动请求管理员权限（点击"是"）
3. 脚本运行后，托盘区会出现 AutoHotkey 图标

### 3️⃣ 开始使用

1. 复制需要处理的文本到剪贴板（Ctrl+C）
2. 按下对应的快捷键
3. 在目标位置粘贴（Ctrl+V）

### 🎯 文本整理核心功能（Win+字母）

#### 基础整理

| 快捷键  | 功能           | 说明                         |
| ------- | -------------- | ---------------------------- |
| `Win+V` | 基础文本规范化 | 修正标点、删除多余空格和换行 |
| `Win+E` | 清理首尾标点   | 删除文本首尾的标点符号       |
| `Win+R` | 移除序号和括号 | 去除行首序号、首尾括号和标点 |
| `Win+Z` | 智能断句       | 在句号和序号后自动换行       |

#### 添加标点

| 快捷键  | 功能             | 效果              |
| ------- | ---------------- | ----------------- |
| `Win+D` | 文本末尾添加句号 | `文本` → `文本。` |
| `Win+F` | 文本末尾添加分号 | `文本` → `文本；` |
| `Win+G` | 文本末尾添加逗号 | `文本` → `文本，` |

#### 智能分段

| 快捷键  | 功能         | 说明                                         |
| ------- | ------------ | -------------------------------------------- |
| `Win+S` | 按句子分段   | 在 `。！？` 处换行                           |
| `Win+;` | 按分号分段   | 在 `:：；;。?？!！` 处换行                   |
| `Win+,` | 按逗号分段   | 在 `，,:：；;。?？!！` 处换行                |
| `Win+-` | 按连字符分段 | 在 `-—:：` 处换行                            |
| `Win+/` | 按顿号分段   | 在 `、，：；` 处换行，适合列表               |
| `Win+?` | 按连接词分段 | 在 `、，：；` 和 `或者、及其、以及` 等处换行 |

#### 添加包裹符号

| 快捷键  | 功能     | 效果                |
| ------- | -------- | ------------------- |
| `Win+Q` | 添加引号 | `文本` → `"文本"`   |
| `Win+B` | 添加括号 | `文本` → `（文本）` |

### 🔢 序号处理（Win+数字）

#### 自动编号

| 快捷键  | 功能          | 说明                           |
| ------- | ------------- | ------------------------------ |
| `Win+0` | 重置计数器    | 将序号计数器归零               |
| `Win++` | 计数器+1      | 序号递增                       |
| `Win+=` | 计数器-1      | 序号递减                       |
| `Win+1` | 添加序号+文本 | 自动添加递增序号，如 `1. 文本` |
| `Win+2` | 仅添加序号    | 仅输出序号，如 `1. `           |

**💡 提示**：连续使用 `Win+1` 会自动递增序号（1. 2. 3. ...），两分钟不操作自动归零。

#### 批量编号

| 快捷键  | 功能             | 说明                                                       |
| ------- | ---------------- | ---------------------------------------------------------- |
| `Win+M` | 为多行添加序号   | 每行开头添加 `1. 2. 3.` 序号，最后一行用句号，其他行用分号 |
| `Win+N` | 行首编号并格式化 | 先按换行分段，再添加序号                                   |

#### 其他功能

| 快捷键  | 功能           | 说明                                 |
| ------- | -------------- | ------------------------------------ |
| `Win+J` | 合并为顿号列表 | 将多行合并为一行，用顿号分隔         |
| `Alt+N` | 提取数字       | 删除所有非数字字符，仅保留数字和换行 |

## 🔧 高级功能说明

### 自动纠错功能

脚本内置了大量 OCR 识别常见错误的自动修正规则：

- ✅ 繁体字 → 简体字（使用 `繁简转换对照表.txt`）
- ✅ 异体字 → 规范字
- ✅ 常见错误：`栽定` → `裁定`、`白勺` → `的`、`不熊` → `不能`
- ✅ 标点修正：全角半角自动转换
- ✅ 中文数字智能转换：
  - 序号：`第一条` → `第 1 条`
  - 分数：`三分之二` → `2/3`
  - 计量：`二十万元` → `20 万元`
  - 百分比：`百分之五` → `5%`

### 智能标点处理

- 汉字前后自动使用全角标点（，。；：）
- 英文、数字前后自动使用半角标点（, . ; :）
- 自动删除多余的空格和换行
- 自动修正括号类型（汉字用全角，英文数字用半角）

## ❓ 常见问题

### Q1: 脚本无法运行？

**A**: 确保安装的是 **AutoHotkey v2** 版本，而非 v1 版本。

### Q2: 快捷键不生效？

**A**:

1. 检查脚本是否正在运行（查看托盘图标）
2. 确保已复制文本到剪贴板
3. 部分快捷键可能与其他软件冲突

### Q3: 如何修改邮箱和手机号？

**A**: 使用文本编辑器打开 `MindMap.ahk`，找到第 11 和 15 行，修改引号内的内容。

### Q4: 如何停止脚本运行？

**A**: 右键点击托盘区的 AutoHotkey 图标，选择"Exit"。

### Q5: 繁简转换不准确？

**A**: 可以编辑 `繁简转换对照表.txt` 文件添加自定义转换规则。
